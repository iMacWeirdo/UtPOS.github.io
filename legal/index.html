1. Purpose and Intent

This software (the "Project") is an independently developed, open-source POS solution created to ensure the continued utility and functionality of Oasys IPadSX hardware.

The primary goal of this Project is interoperability. As the original software (ChefXML / ROD Gold) has reached End-of-Life (EOL) or is no longer supported by the manufacturer, this Project provides a replacement communication protocol to prevent hardware obsolescence and electronic waste.

2. Method of Development (Clean Room)

The communication protocol implemented in this Project was derived through "Black Box" reverse engineering methods, specifically:

  • Network Traffic Analysis: Using industry-standard tools (e.g., Wireshark) to observe data packets sent between the hardware and server on a private, lawfully owned network.
  • Independent Implementation: No original source code, binaries, or proprietary design documents from Select Electronics were accessed, decompiled, or copied. All code in this Project is original work.

3. Legal Protections & Fair Use

We believe this Project is protected under several key legal doctrines (as of 2026):

  • 17 U.S.C. § 1201(f) (DMCA): Provides a specific exemption for the circumvention of technological measures for the sole purpose of identifying and analyzing those elements of a program that are necessary to achieve interoperability with an independently created program.
  • Nominative Fair Use: References to "ChefXML," "ROD Gold," "Oasys," and "Select Electronics" are used strictly for descriptive purposes to identify compatible hardware and legacy systems. Such use is protected as it is necessary to inform the user about the software's application.
  • Right to Repair: This project aligns with the spirit of modern "Right to Repair" exemptions, enabling owners of legally purchased hardware to maintain its utility after manufacturer support has ceased.
